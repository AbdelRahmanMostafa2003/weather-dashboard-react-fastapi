# Weather Dashboard

This project is a simple weather dashboard built with React and Vite.

## Features
- Search for a city and view current weather from OpenWeatherMap
- See a mock 5-day forecast section with placeholder data
- Save favorite cities and keep recent search history
- Toggle between light and dark mode

## Note on Forecast Data
The current weather is retrieved from OpenWeatherMap. The forecast section currently uses placeholder data because of API limitations.
